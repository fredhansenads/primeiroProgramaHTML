# Minha Primeira Página HTML

Página de teste criada durante o estudo do curso de HTML.

## O que é

Uma página simples, com o conteúdo "Bem-vindo à minha primeira página!" e o texto "Este é um exemplo simples de uma página HTML." Serve como primeiro exercício prático de estruturação de HTML.

## Como usar

Basta abrir o arquivo `.html` no navegador. Não precisa de instalação nem servidor.
